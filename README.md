# 📦 Stock Maintenance System

## 📌 Description
This project is a Stock Maintenance System used to manage inventory, track stock levels, and handle product data efficiently.

## 🚀 Features
- Add new products
- Update stock
- Delete products
- View inventory

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- index.html
- style.css
- script.js

## ▶️ How to Run
1. Download the project
2. Open index.html in browser
